# Olympic Games Data Lake

## Estrutura do Projeto

Este data lake contém dados históricos e atuais dos Jogos Olímpicos.

## Camadas de Dados

- **Raw**: Dados originais em CSV
- **Bronze**: Dados convertidos para Parquet e integrações iniciais
- **Gold**: Análises e visualizações finais

## Datasets Incluídos

1. Dados históricos dos Jogos Olímpicos (1896-2020)
2. Dados das Olimpíadas Paris 2024
